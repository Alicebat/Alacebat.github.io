在本地搭建Hexo博客

为什么选择Hexo

  • 使用Markdown语法写博客，可免去把文本转换成HTML的痛苦，因此对Markdown支持得好，是下面一切选择的前提。支持Markdown的博客工具之中，比较入眼的有ghost, Jekyll和hexo。

hexo获得的关注虽不如前两者，但它有几点很讨人喜欢：

  • 依赖少（仅依赖node），易于安装
  • 台湾人写的，不用担心对中文支持不好
  • 对程序员友好，要是愿意折腾还是有的折腾的
    基于以上原因，我最终选择了hexo作为维护博客的工具
